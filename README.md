## Some things I have built

### MADIS - Multi-Agent Disaster Surveillance

A multi-agent RL environment where drone agents coordinate in a disaster zone with partial observability, false-positive reports, severity-aware rewards, and an LLM coordinator trained using GRPO.

Built for the **Meta x Hugging Face x Scaler Hackathon**, where I was a finalist in the **top 1.5% of 52,000+ participants**.

**Tech:** Python, Qwen3, GRPO, multi-agent RL, Hugging Face

---

### TalentFlow - AI Voice Recruitment Screener

A full-stack recruitment screener that triggers outbound AI voice calls, screens candidates autonomously, and sends structured results like fit score, transcript, and notice period to a live dashboard.

**Tech:** React, Express.js, SQLite, Bolna API, webhooks, polling

---

### EasyPizzy - GPT-Powered Grocery Assistant

A grocery assistant that takes natural language inputs like budget, health goals, dietary restrictions, and preferences, then recommends products across grocery platforms with availability filtering and substitution logic.

**Tech:** React, FastAPI, OpenAI GPT, recommendation logic, payment flow

---

### Intelligent Study Platform

A full-stack AI-powered study platform with real-time collaboration, quiz generation, summarization, authentication, and modular backend architecture.

**Tech:** React, TypeScript, Vite, Tailwind, Express.js, PostgreSQL, Drizzle ORM, OpenAI API, WebSockets
